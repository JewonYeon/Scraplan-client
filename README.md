# Scraplan



Scraplan은 지도를 기반으로 **해당 장소의 즐길거리**를 큐레이션 형식으로 제공하고 <br>
해당 큐레이션을 이용해 **나만의 일정**을 작성할 수 있는 서비스입니다.

비회원의 경우 일정 작성, 다른 사람의 일정 구경까지 기능을 체험해 볼 수 있으며<br>
해당 일정을 저장하거나 다른 사람에게 공유하고 싶은 경우는 회원만 가능합니다.
